The Twelve-Factor Design Ops
=====================

Source for the content app running at: https://12factor.design/

Strongly inspired in https://12factor.net for DevOps

Development
-----------

    bundle install
    heroku local:start
    open http://localhost:5000

Production deploy
-----------------

    heroku create
    git push heroku master
    heroku open
